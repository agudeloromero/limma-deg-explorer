Methods

Differentially expressed genes were identified from limma-voom output (Ritchie et al., 2015, Nucleic Acids Research; Law et al., 2014, Genome Biology) using a threshold of adjusted P-value ≤ {{ deg_pval }} and |log₂FC| ≥ {{ deg_lfc }} (Benjamini–Hochberg correction). Genes were ranked by sign(log₂FC) × −log₁₀(adj.P.Val), following the pre-ranked GSEA convention (Subramanian et al., 2005, PNAS).

Functional enrichment analysis was performed across three Gene Ontology domains (Biological Process, Molecular Function, Cellular Component) using gseapy v (Fang et al., 2023) via the Enrichr web API (Chen et al., 2013), with results filtered by adjusted P-value ≤ {{ enr_pval }}.

Difference from R implementation: the R version uses clusterProfiler with org.Mm.eg.db (offline, mouse annotation database), which does not require an internet connection. The Python version queries the Enrichr API and requires internet access. Gene set names differ between implementations (clusterProfiler: GO ontology codes BP/MF/CC; gseapy/Enrichr: GO_Biological_Process_2023 etc.).
